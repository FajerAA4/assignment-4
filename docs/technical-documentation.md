## 2. Architecture

The architecture follows a client-side application model:

- HTML renders structure  
- CSS controls layout, visuals, and animations  
- JavaScript handles logic and state  
- APIs provide live dynamic content  
- localStorage stores settings and session-related values  

---

## 3. Features

### Dark Mode
Theme switching is implemented using
class toggling and persistence using `localStorage`.

### Filtering
Projects are filtered dynamically using arrays and conditions.

### APIs
External APIs are fetched asynchronously using `fetch()` and `async/await`.
