# My Professional Portfolio (Assignment 4)

## Project Description

This project is a fully functional personal portfolio website built to showcase academic projects, technical skills, and web development capabilities.

The application demonstrates practical experience in:

- Front-end web development
- Responsive design
- API integration
- JavaScript interactivity
- Responsible AI usage

The website includes dynamic features such as live weather data, quotes, project filtering, theme memory, and user interaction features.

---

## Features

| Feature | Description |
|---------|-------------|
| Dark Mode | Saves theme preference using `localStorage` |
| Greeting System | Greeting changes based on time of day |
| Project Filtering | Filter by category and level |
| Sorting | Sort by date and name |
| Search Bar | Search projects dynamically |
| Quote API | Displays random quotes |
| Weather Widget | Live weather display |
| Login Simulation | UI-based login state control |
| Name Memory | Stores visitor name |
| Session Timer | Tracks visit duration |
| Responsive Layout | Works across screen sizes |
